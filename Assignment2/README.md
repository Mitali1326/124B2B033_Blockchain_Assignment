**Assignment 2: Smart Contract Deployment on Polygon**

 **Objective**

The objective of this assignment is to deploy a Solidity smart contract on the Polygon Amoy Testnet using Remix IDE and MetaMask, and to understand real-world blockchain deployment and transaction verification.

**Deployment Steps**

Step 1: Open Remix IDE
Go to https://remix.ethereum.org
Create a new file named contract.sol
Paste your smart contract code

Step 2: Compile Contract
Navigate to the Solidity Compiler tab
Select compiler version 0.8.x
Click Compile contract.sol

Step 3: Configure MetaMask
Install MetaMask browser extension
Add Polygon Amoy Testnet using the network details above
Switch to the Polygon Amoy network

Step 4: Connect MetaMask to Remix
Go to Deploy & Run Transactions
Select Environment → Injected Provider - MetaMask
Connect your MetaMask wallet

Step 5: Deploy Contract
Click Deploy
Confirm the transaction in MetaMask
Wait for the contract to be successfully deployed

Step 6: Verify Deployment
Copy the deployed contract address
Open Polygon Explorer: https://amoy.polygonscan.com
Paste the contract address to view transaction details
